Kitty Shortcuts — Dev Mode

Clipboard
  Ctrl+Shift+C   → Copy
  Ctrl+Shift+V   → Paste

Tabs & Windows
  Ctrl+Shift+T   → New tab
  Ctrl+Shift+W   → Close tab
  Ctrl+Shift+←→  → Switch tabs
  Ctrl+Shift+↑↓  → Move tab position

Splits
  Ctrl+Alt+Enter → New window
  Ctrl+Alt+V     → Horizontal split
  Ctrl+Alt+H     → Vertical split

Dev Tools
  Ctrl+Shift+F   → fzf
  Ctrl+Shift+G   → lazygit
  Ctrl+Shift+L   → lazydocker
  Ctrl+Shift+R   → ranger
  Ctrl+Shift+E   → code .
  Ctrl+Alt+L     → docker logs -f

Terminal Goodies
  Ctrl+Shift+D   → dust
  Ctrl+Shift+L   → lsd
  Ctrl+Shift+B   → bat kitty.conf

Kitty Actions
  Ctrl+Shift+X   → hints (clickable links)
  Ctrl+Shift+Z   → clipboard kitten
  Ctrl+Shift+Q   → send ESC
